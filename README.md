## How to run

1. Clone this repository
2. cd folder
3. Build database
```console
> dotnet ef database update
```

4. Run from Visual Studio or cmd line
```console
> dotnet watch
```
5. Can be access at http://localhost:5001/api/values (depends on values in launchSettings.json)
